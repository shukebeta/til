# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Is

A personal tech notes journal. Each note is a self-contained article distilled from a Q&A session.

## Structure

- `notes/<yyyy-mm>/<yyyy-mm-dd>_<topic>.md` — notes organized by month
- `README.md` — project description

## Conventions

- File names: `yyyy-mm-dd_topic.md` — no spaces or special characters. Chinese characters allowed.
- Note format: markdown with a `# Title`, body prose, and a `## Key Points` section at the end with 3-5 bullets.
- Language: match whatever the user spoke in the session (English or Chinese). Technical terms always in English.
- Notes must be self-contained — readable without conversation context.

## The /record Skill

Use `/record` at the end of a session to capture the Q&A as a new note. The skill handles topic extraction, formatting, and monthly directory creation.
